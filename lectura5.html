GITHUB

Es un servicio para alojamiento de repositorios de software, gestionados por el sistema de control de versiones GIT.

El GITHUB es muy particular, es un sitio WEB que usa el GIT para ofrecer a la comunidad de desarrolladores repositorios de software, en definitiva GITHUB es un sitio que nos permite compartir el código de una manera muy fácil y al mismo tiempo darle popularidad a la herramienta de control de versiones.

Teniendo en cuenta que GIT es un proyecto comercial, en internet existen sitios populares que aplican la misma fórmula, uno de los más reconocidos es BitBucket.

Una cualidad de GITHUB es que es totalmente gratuito y en estos momentos hay varios millones de usuarios trabajando con la herramienta.

Tengamos en cuenta que GITHUB se ha convertido en una herramienta para las empresas que revisan los repositorios de GITHUB y así mirar los proyectos y contribuciones que los programadores aportan, por lo tanto es importante mantener un perfil activo.
