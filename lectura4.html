GIT

Es un software de control de versiones diseñado para la eficiencia y la confiabilidad del mantenimiento de versiones de aplicaciones, cuando estas tienen un gran número de archivo de código de fuente.

Algunos de los beneficios que nos aporta la herramienta GIT:

    • Tener control sobre cómo ha cambiado un proyecto a lo largo del tiempo.
      Volver al inicio de forma instantánea.
      Tener el control de las versiones a través de etiquetas, como ejemplo versión 1.0, versión 1.1, versión 1.2, etcétera.
      Saber exactamente de que trataba cada una de ellas y la diferencia entre las anteriores versiones de nuestro programa.
      En la parte de seguridad nos da la confianza de que las estructuras internas de los datos tienen una firma de seguridad, la cual nos informa cualquier cambio que tenga el código.
