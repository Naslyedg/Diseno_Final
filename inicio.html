Objetivo

Como principal objetivo vemos la profundización de temas como HTML5 y CSS3, pasando por una explicación de cómo utilizar GIT y GITHUB, grandes herramientas para el desarrollo del control de versiones en las que podemos trabajar de manera grupal en un desarrollo sin temor a no incluir los aportes de algún integrante del grupo.

Curso de Diseño de sitios WEB

¿De qué se trata?

Es un curso perteneciente al programa de ingeniería de Sistemas de la UNAD, conformando la parte electiva de dicha carrera.

Dentro del curso vemos el desarrollo de temáticas como la utilización de sistemas de control de versiones como lo es GIT y por ende GITHUB, así mismo vemos el uso en particular de lenguajes como HTML5 y CSS3.

El curso consta de 3 créditos académicos y se distribuye en 3 unidades de desarrollo de actividades las cuales son:

    • 1. Introducción al diseño de sitios WEB
      2. HTML5
      3. CSS3
    • Competencias que se desarrollan mediante el curso:

    • Satisfacer necesidades de los usuarios con los diferentes desarrollos de páginas WEB.
      Actualización de páginas WEB.
      Conocimiento y operación en lenguajes como HTML5 y CSS3.
      Administración y operación de sistemas de control de versiones como GIT y GITHUB.
      Uso y reconocimiento de varias metodologías de Diseño WEB.
